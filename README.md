## AWE 
AWE is a tool for testing system security using access codes during testing.

## Usage
Clone the repository:
```
git clone https://github.com/ffoster007/AWE.git
```

Run the application:
```
awe [options]
```

## For Developers
To build and install the application from source, use the following commands:
```
make uninstall && make install
```
